KEY FINDINGS
================================================================================

1. ANIMATED CONTENT VS CROSSFADE
   - Use AnimatedContent for finance apps (Multiple states)
   - Use Crossfade for simple transitions (Loading â†’ Content)
   - Different animations for different state transitions improve UX
   - AnimatedContent with SizeTransform handles layout changes smoothly

2. SKELETON LOADING BEST PRACTICES
   - Shimmer animation duration: 1500ms (hypnotic, not distracting)
   - Minimum display time: 150ms (prevents flickering for fast loads)
   - Layout consistency: Skeleton must match loaded content exactly
   - Progressive loading: Stack skeletons (balance â†’ transactions)

3. STAGGER ANIMATIONS
   - Sweet spot: 50-100ms between items
   - Total stagger: 300-400ms max (don't make users wait)
   - Key requirement: Use unique keys in LazyList
   - Critical: Hoist state outside LazyColumn (prevent retrigger on scroll)

4. PERFORMANCE TARGETS
   - Android 60Hz: < 16ms per frame (achievable)
   - iOS 120Hz: < 8.33ms per frame (achievable, tested on iPhone 15 Pro)
   - Animation duration: 250-400ms for finance (feels responsive)
   - Frame drops: Target 0 (achievable with graphicsLayer)

5. COMPOSE MULTIPLATFORM 1.9.3
   - iOS animations: 6% faster with AnimatedVisibility
   - List scrolling: 9% faster with LazyGrid
   - Shimmer effect: 360% faster (8.8s â†’ 2.4s CPU time)
   - iOS status: STABLE (production-ready since October 2024)
   - Platform-specific code: NOT NEEDED (same code works everywhere)
